About IP Tracker

Objective

To develop a web-based tool that analyzes the reputation of IP addresses by scraping data from multiple sources and applying a custom algorithm to categorize IPs as malicious, safe, or suspicious.

Abstract

The IP Tracker project provides a unified platform for IP reputation analysis. It gathers threat intelligence from sites like VirusTotal, AbuseIPDB, and Shodan using scraping (not APIs) and combines the data with a proprietary algorithm for final classification. The system is designed to be lightweight, fast, and informative for cybersecurity purposes. Future upgrades include ML-based classification and history tracking.

How It Works

Users enter an IP address, and the backend scrapes results from major threat intelligence platforms. These are parsed, analyzed, and scored using preset rules and optional ML models. The final status is displayed with a detailed report.
